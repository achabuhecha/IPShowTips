  IP 悬浮窗（IPShowTips）— 使用说明
  版本：v1.0.0
═══════════════════════════════════════

【软件简介】
  一款 Windows 桌面小工具，在屏幕右下角以悬浮球形式
  显示你的公网 IP 和归属地，并在 IP 变化时弹出提醒。

───────────────────────────────────────
  一、安装与启动
───────────────────────────────────────

  方式一：直接运行（推荐）
    1. 双击 IPShowTips.exe
    2. 屏幕右下角出现圆形「IP」悬浮球即表示运行成功

  方式二：Python 源码运行
    1. 安装 Python 3.8 或以上版本
    2. 打开命令行，进入程序目录
    3. 执行：pip install -r requirements.txt
    4. 执行：python ip_widget.py

  提示：若任务栏托盘看不到图标，请点击右下角 ^ 展开隐藏图标。

───────────────────────────────────────
  二、基本操作
───────────────────────────────────────

  ┌──────────────────┬────────────────────────────┐
  │ 拖动悬浮球       │ 按住左键拖动，调整位置     │
  │ 单击悬浮球       │ 展开 IP 详情面板           │
  │ 鼠标移开面板     │ 自动收起为悬浮球           │
  │ 右键悬浮球       │ 打开功能菜单               │
  │ 双击托盘图标     │ 重新显示悬浮球             │
  └──────────────────┴────────────────────────────┘

───────────────────────────────────────
  三、右键菜单
───────────────────────────────────────

  · 最小化到托盘 — 隐藏悬浮球，在托盘继续监控
  · 退出程序     — 完全关闭软件

  托盘图标右键菜单：
  · 显示悬浮球
  · 最小化到托盘
  · 退出

───────────────────────────────────────
  四、面板信息说明
───────────────────────────────────────

  展开面板后可看到：
  · IP 地址
  · 归属地（省份、城市、运营商）
  · 最后刷新时间（最近一次查询成功的时间）
  · 版本号（面板底部显示，如 v1.0.0）

  程序每 10 秒自动刷新一次 IP 信息。

───────────────────────────────────────
  五、IP 变化提醒
───────────────────────────────────────

  当检测到公网 IP 与上次不同时，会自动弹出提醒框，显示：
  · 原 IP
  · 新 IP
  · 归属地

  点击「知道了」关闭提醒。
  首次启动获取 IP 时不会弹窗，仅在 IP 发生变化时提醒。

───────────────────────────────────────
  六、其他说明
───────────────────────────────────────

  · 默认启动位置：屏幕右下区域
  · 悬浮球可拖动，但不会拖出屏幕边界
  · 靠近屏幕边缘展开面板时，会自动向内侧展开
  · 需要联网才能获取 IP 信息
  · 本工具仅支持 Windows 系统

───────────────────────────────────────
  七、常见问题
───────────────────────────────────────

  问：单击悬浮球没反应？
  答：请确保是「轻点」而非拖动。拖动用于移动位置，
      轻点一下才会展开面板。

  问：如何彻底退出？
  答：右键悬浮球或托盘图标，选择「退出」。

  问：exe 被杀毒软件拦截？
  答：PyInstaller 打包的程序可能被误报，添加信任即可，
      或使用源码自行打包。
