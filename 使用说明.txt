  IP 悬浮窗（IPShowTips）— 使用说明
  版本：v1.0.0
═══════════════════════════════════════

【软件简介】
  一款 Windows 桌面小工具，在屏幕右下角以悬浮球形式
  显示你的公网 IP 和归属地，并在 IP 变化时弹出提醒。

  本项目分两个目录，功能相同：
  · autohotkey/  — 轻量版（推荐）
  · python/      — Python 版

───────────────────────────────────────
  一、安装与启动
───────────────────────────────────────

  方式一：直接运行 exe（推荐）
    · 轻量版：autohotkey\dist\IPShowTips_ahk.exe
    · Python 版：python\dist\IPShowTips_py.exe
    屏幕右下角出现圆形「IP」悬浮球即表示运行成功

  方式二：Python 源码运行
    1. 安装 Python 3.8 或以上版本
    2. 进入 python 目录
    3. 执行：pip install -r requirements.txt
    4. 执行：python ip_widget.py

  方式三：AutoHotkey 脚本运行（开发者）
    1. 安装 AutoHotkey v2
    2. 进入 autohotkey 目录，双击 IPShowTips.ahk

  提示：若任务栏托盘看不到图标，请点击右下角 ^ 展开隐藏图标。

───────────────────────────────────────
  二、基本操作
───────────────────────────────────────

  ┌──────────────────┬────────────────────────────┐
  │ 拖动悬浮球       │ 按住左键拖动，调整位置     │
  │ 单击悬浮球       │ 展开 IP 详情面板           │
  │ 鼠标移开面板     │ 自动收起为悬浮球           │
  │ 右键悬浮球       │ 打开功能菜单               │
  │ 双击托盘图标     │ 重新显示悬浮球             │
  └──────────────────┴────────────────────────────┘

───────────────────────────────────────
  三、打包编译
───────────────────────────────────────

  AutoHotkey 版：进入 autohotkey 目录，运行 build.bat
  Python 版：    进入 python 目录，运行 build.bat

───────────────────────────────────────
  四、其他说明
───────────────────────────────────────

  · 默认启动位置：屏幕右下区域
  · 悬浮球可拖动，但不会拖出屏幕边界
  · 需要联网才能获取 IP 信息
  · 本工具仅支持 Windows 系统
  · 两个版本功能相同，AHK 版体积更小，推荐日常使用

───────────────────────────────────────

  数据来源：太平洋电脑网 IP 查询接口
